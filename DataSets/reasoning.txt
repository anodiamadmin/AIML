Q3 D1:
How many points have high leverage? rule of thumb -> high leverage if its leverage value is ≥3×p/N.
Code:
sum(hatvalues(model_D1) >= 3*length(coef(model_D1))/nrow(colony_counts))

LHS:
sum: Counts the number of TRUE values
hatvalues(model_D1): Computes leverage values (hᵢ) for each observation in Model D1. Measure how far each point’s predictors are from the mean of predictors
RHS:
3 * length(coef(model_D1)) / nrow(colony_counts): Computes the threshold = 3p/N (rule of thumb for high leverage)
coef(model_D1): Extracts the model coefficients (including intercept)
length(coef(model_D1)): Gives p, the number of parameters in the model
nrow(colony_counts): Gives N, the number of observations (rows in dataset)

Q5 E1: leave-one-out CV lambda=3 -> estimate test MSE

This code manually performs leave-one-out cross-validation by leaving out one point at a time, fitting the ridge model on the remaining data, predicting the omitted point, squaring its error, and averaging those errors.

(mean(sapply(1:nrow(x), function(i) {
  fit <- glmnet(x[-i, ], y[-i], alpha = 0, lambda = 3)
  (y[i] - predict(fit, newx = x[i, , drop = FALSE], s = 3))^2
}))

1:nrow(x) -> Iterates through each observation, treating each one once as the left-out test point
sapply(..., function(i) { ... }) -> Repeats the leave-one-out process for every observation and stores the squared prediction error each time
x[-i, ] and y[-i] -> Use all observations except the i-th one as the training set
glmnet(..., alpha = 0, lambda = 3) -> Fits the ridge regression model with λ = 3 on the training data
x[i, , drop = FALSE] -> Selects the left-out observation as a 1-row matrix so predict() works correctly
predict(fit, newx = x[i, , drop = FALSE], s = 3) -> Predicts the response for the omitted observation using the fitted ridge model
(y[i] - predict(...))^2 -> Computes the squared test error for that left-out point
mean(...)
